# Investor Behavior Analysis

## Status

🚧 In Development

## Overview

Machine Learning project using Brazilian CVM public data to analyze investor behavior, identify patterns and predict risk profiles.

The project combines clustering, time series forecasting, predictive modeling and AI-generated executive reports.

---

## Features

* Investor segmentation with K-Means
* Time series forecasting
* Risk profile prediction
* AI-generated executive reports with Claude API
* Interactive dashboard with Streamlit

---

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Plotly
* Prophet
* Claude API

---

## Goals

* Analyze investor behavior patterns
* Predict investor risk profiles
* Generate business insights automatically
* Build an interactive financial dashboard
